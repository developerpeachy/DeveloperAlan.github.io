Visualize and Map location data to find trends

Add another element to the typical spreadsheet data by visualizing the location data on a map. Maps can be created with filters to allow for even better analysis. Mapping, filtering, and viewing data is fast and easy.

The example above shows cities with high median incomes and filtering enabled by a range of the city's population.

EasyMapMaker allows up to 12 filters to be selected from the advanced maps tab. Up to 200 items can be filtered on. Excel makes adding custom ranges easy, and EasyMapMaker allows the ranges to be hidden from the resulting data.

Data can also be grouped (or categorized) and displayed in a pie chart when clustering is enabled. Data is automatically grouped into ranges if there are more than 10 categories. Categorizing data on the map is as easy as setting the group by tab from the map options.
