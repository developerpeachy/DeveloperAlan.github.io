Mapas avanzada

¿Alguna vez ha querido traer filtrar datos en Excel que le Mapas? Nuestros "Mapas avanzadas" le permiten hacer precisamente eso. La pestaña Avanzada Map permite la misma funcionalidad que Easy "Mapas", además de más. Personalizar el mapa añadiendo sus propias imágenes y logotipo. Si desea ver los datos que se contienen en una región del mapa, la función de exportación de datos proporciona esto.

Mira el video para hacer un mapa con los filtros, los pernos de encargo, un logotipo, exportación de datos, direcciones de conducción, y un filtro de ubicación.

Mapas avanzadas permiten una mayor personalización:

  1. Añadir imágenes personalizadas
  2. Añadir filtros (Selección múltiple, de selección simple, cuenta de artículos)
  3. Ocultar datos

Añadir imágenes personalizadas (Ver mapa de ejemplo )

Añadir una imagen personalizada o el logotipo de su mapa subiéndolos. Puede cargar hasta 50 imágenes en 1 hora con un tamaño de hasta 10 KB cada uno. Para utilizar varias imágenes, el grupo necesita ser ajustado. Las imágenes se pueden asignar a los diferentes grupos (o categorías). La imagen de abajo es un ejemplo de configuración de un grupo (que se utilizará con las imágenes en la foto superior).

Añadir filtros con un simple clic.

Ocultar elementos que no quieren que los usuarios vean

Our Mapas avanzada le permiten tener datos de búsqueda y que se pueden filtrar en el mapa pero no se muestra a los espectadores del mapa. detalles de la dirección ahora se pueden ocultar fácilmente, o artículos que se filtran o agrupadas en sólo están pueden ser visibles con los datos de origen y oculto a los usuarios. Esto es tan fácil como seleccionar los elementos a ocultar. Por defecto, todos los elementos visibles.

El mapa de arriba es una muestra usando grupos y filtros

Mapas avanzados también incluyen la opción de ocultar todas las cabeceras de la etiqueta para la descripción del marcador por la de-seleccionar la casilla de verificación "mostrar la etiqueta". Esta opción debe utilizarse con precaución a menos que los datos es evidente para el usuario.
