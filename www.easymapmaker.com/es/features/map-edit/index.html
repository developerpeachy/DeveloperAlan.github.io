Sencilla edición de mapas

¿Quieres actualizar el mapa con los datos modificados? Si se ha introducido una dirección de e-mail al guardar el mapa, habrá habrá una URL para ver el mapa y otro URL para acceder a una página para editar el mapa.

Características Editar página

La página de edición permite a los siguientes cambios en el mapa:

  • Modificar el mapa de alias para una fácil de recordar URL
  • Cambiar el nivel de seguridad mapa
  • Modificar el título del mapa y Descripción
  • Eliminar o mostrar los datos situados debajo del mapa
  • Actualización de los datos en el mapa

La actualización de los datos de los mapas

La página de Cartografía es un gran lugar para agregar datos al mapa que anteriormente no podían ser geo-codificados debido a problemas de tiempo de espera. Es también donde se pueden introducir los datos modificados a partir de la hoja de cálculo. No hay necesidad de copiar los datos, basta con pegar los datos de hoja de cálculo con las cabeceras que coincidan con los datos existentes. Las coordenadas de mapeado (latitud y longitud) se almacenan en caché, por lo que tendrán que ser codificadas geo sólo los nuevos datos.

Siga los pasos a continuación para actualizar su mapa:

  1. Copiar los datos de hoja de cálculo modificados desde el programa de hoja de cálculo
  2. Haga clic en "Re-Map"
  3. Guarde el mapa después de hacer cualquier modificación
  4. Si hay más de 250 artículos nuevos, actualizar la página de edición después de guardar y repetir el paso 1.
