Häufig gestellte Fragen und Antworten (FAQ)

Wie viele Adressen kann ich zu einer Karte hinzufügen?
Sie können über 10.000 einzigartigen Orten pro Tag zur Karte hinzufügen. Es wird Stunden dauern, sie zu geocodiert, aber es ist machbar. Wenn Sie bereits die Längen- und Breitenkoordinaten haben können die Daten in weniger als 5 Sekunden hinzugefügt werden.
Wie kann ich mehr als 250 Einzelteile zu einer Karte hinzufügen?

Sie können Tausende von Standorten täglich zur Karte hinzufügen. Die Grenze ist mindestens 10.000 Stellen.

Nur 250 Adressen gleichzeitig verarbeitet werden können, nach 250 Standorten Geokodierung auf ein Schneckentempo geht.

Schauen Sie sich die Beispiel-Video, wie Hunderte oder Tausende von Artikeln abzubilden.
How to make a map with over 500 locations Make a map with easymapmaker.com with over 500 locations from your excel or other spreadsheet data. Go to http://www.easymapmaker.com to make your own map.

Sobald 250 Adressen fertig sind, speichern Sie Ihre Karte und einen Link zum Massen geocode angezeigt. Klicken Sie auf diesen Link, und die Orte für Sie automatisch in Chargen von 250 und automatisch gespeichert geo-codiert werden. Nachdem alle die Adressen werden geo-codiert wird der Edit-Seite geladen werden. Mit den kostenlosen Google-Lizenz kann es eine Stunde mehr als 5.000 Adressen zu verarbeiten.

Was ist los, warum ist Ihr Service kostenlos?
Unser Service ist 100% kostenlos. Wir zahlen nicht google etwas ihre API verwenden. Unser Team macht Geld durch ihren Tag Arbeitsplätze. Wenn Sie spenden möchten, können Sie.
Kann ich die EasyMapMaker Logo auf meiner Karte ersetzen?
Ja. Fügen Sie den folgenden Parameter auf der Karte: logo = <kodierte URL>. Sie können Ihre URL mit unserem kodieren url-Encoding-Tool .

Ein Beispiel ist:

http://www.easymapmaker.com/map/58890adb2af63457f31769e607d7501f?logo=http%3A%2F%2Fwww.easymapmaker.com%2Fcluster_images%2Fsemi_opaq%2F45_blue.png

Eine weitere Möglichkeit, ein individuelles Logo hinzuzufügen, ist erweiterte Karten zu verwenden und ein Logo im Anpassungsbereich hochladen.

Der Standardbildgröße 45x45 Pixel und der Austausch sollte idealerweise die gleiche sein.

Kann ich die Schriftart zu ändern oder individuelle Styling meiner Karte hinzufügen?
Ja. Fügen Sie den folgenden Parameter auf der Karte: css = <kodierte URL>. Sie können Ihre URL mit unserem kodieren url-Encoding-Tool. Ein Beispiel ist: http://www.easymapmaker.com/map/58890adb2af63457f31769e607d7501f?css=http%3A%2F%2Fwww.easymapmaker.com%2Fcssoverride%2Fhideimg.css
Wie kann ich eine vorhandene Karte aktualisieren oder löschen?
Wenn Sie eine E-Mail-Adresse eingegeben beim Speichern Sie Ihre Karte. Es sollte eine URL enthalten sind, um die Karte zu bearbeiten. Die Karte kann auch von dieser Seite entfernt werden. Manchmal werden die E-Mails der Spam oder Junk-Mail-Ordner zu gehen, so dass Sie auch dort überprüfen können. Wenn Sie eine Liste aller Ihre Karten für ein E-Mail-Adresse wollen, gehen Sie zu http://www.easymapmaker.com/findMyMaps
Machst du Anpassungen?
Nein, wir werden nichts benutzerdefinierte tun. Wir werden Funktionen basierend auf Anforderungen hinzufügen. Es gibt eine Menge von Funktionen, die wir schreiben wollen, aber es wurden alle Anträge für sie nicht gewesen. Fühlen Sie sich frei Merkmale zu ersuchen, für unsere Kontaktseite.
Wie kann ich meine Daten auf Google Maps sehen?
Wenn Sie das kml erzeugt, können Sie es auf einer Google-Karte importieren (hosted by Google).
Es gibt ein Problem mit _fill_in_the_blank Funktion (Es ist ein Fehler).
Bitte lassen Sie uns wissen über unsere Kontaktseite wenn etwas nicht funktioniert. Manchmal wird Google eine Änderung vornehmen und etwas brechen; manchmal wir eine Änderung vornehmen und etwas brechen, und ein anderes Mal etwas einfach nicht funktioniert wie es soll.
Gibt es eine Möglichkeit nicht eine Liste von Daten unter der Karte zeigen?
Ja, Sie können die Daten auf dem unteren Rand der Seite ausschalten. Wenn Sie auf der Bearbeitungsseite gehen (in einer E-Mail nach dem Speichern die Karte gesendet werden soll), gibt es eine "Map-Layout-Seite. Sie haben die Möglichkeit, die Daten zeigt, unterhalb der Seite zu deaktivieren.
Kann ich eine Karte ohne eine Adresse zu tätigen?
Ja. Wenn Sie einen Breitengrad und Längengrad Standort haben, können diese schnell abgebildet werden. Wenn Sie nur einen Teil einer Adresse, wie beispielsweise eine Postleitzahl haben, oder Stadt, Land, dann können diese ebenfalls abgebildet werden.
Welche Länder werden unterstützt?
Google Maps unterstützt die gefunden Länder auf ihrem Kartenabdeckung Seite.
Kann ich ziehen Koordinaten aus einer Datenbank, die eine dynamische Karte zu schaffen?
Nein. Zu diesem Zeitpunkt unserer Lösung unterstützt keine Daten ziehen aus einer Datenbank.

Machen Sie Ihre Karten Arbeit auf mobilen Geräten und Tablets?
Ja. Die meisten mobilen und Tablet-Geräte werden unterstützt. Allerdings sind einige Funktionen, die nicht vollständig auf allen Geräten unterstützt.

Mobile / Tablet-Karten sind auch in der Lage Standort-Dienste zu verwenden, wenn aktiviert.

Erstellen und Bearbeiten von Karten können auf mobilen und Tablet-Geräten durchgeführt werden, aber es wird nicht empfohlen und nicht unterstützt.

Kann ich Sie kaufen ein Bier oder einen Kaffee?
Wir nehmen jetzt Spenden, aber es ist nicht notwendig.
Wie kann ich verschiedene Farben für verschiedene Einzelteile in meinem Datensatz zugeordnet werden, so dass sie auf der Karte identifiziert werden.?
Die Farben werden automatisch auf der Basis der Säule zugeordnet Sie als Gruppe gesetzt. Es gibt bis zu 10 (20 für erweiterte Karten) verschiedenen Farben, die verwendet werden. Wenn eine Spalte mehr als 10 (20 für erweiterte Karten) unterschiedliche eindeutige Werte hat, werden sie zusammen in Bereichen gruppiert werden. Es empfiehlt sich, den Bereich selbst zu definieren und verwenden, die für die Gruppe (wenn Sie mehr als 10 eindeutige Werte in einer Spalte haben).
Wie kann ich Tortendiagramme meine Karte?
Kreisdiagramme automatisch aktiviert werden, wenn Sie auf Clustering drehen und eine Gruppe aus der Karte festgelegt (siehe die oben genannte Frage für Gruppen).
Wie lange wird mein Karte aktiv bleiben?
Karten werden aktiv bleiben, es sei denn sie wurden nicht modifiziert oder in den letzten 90 Tagen gesehen. Sie können jederzeit vor diesem von der Person gelöscht werden, die die Karte gemacht.
Wie kann ich meine Karte zu drucken, es als Bild oder PDF-Datei speichern?

Um Ihre Karte als PDF-Datei speichern, verwenden Sie Google Chrome oder Safari die Karte auszudrucken und als PDF-Datei speichern, anstatt sie zu drucken.

Sie können, indem sie einen Bildschirmausdruck Ihrer Karte Ihre Karte als Bild speichern (Alt + Druck Bildschirm normal funktioniert, oder versuchen Sie Strg + Print Screen). Sobald Sie einen Screenshot haben, sollten Sie in der Lage sein, es in ein Bildbearbeitungsprogramm einzufügen, wie paint.net. Die Größe der Karte kann als Web-Browser-Fenster so groß sein, wenn Sie es nicht einbetten.
