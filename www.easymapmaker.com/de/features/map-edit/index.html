Einfache Karte Editing

Möchten Sie Ihre Karte mit modifizierten Daten zu aktualisieren? Wenn eine E-Mail-Adresse beim Speichern der Karte eingetragen haben, wird es eine URL sein, die Karte und andere URL zu schauen auf eine Seite zuzugreifen, die Karte zu bearbeiten.

Bearbeiten Seite Merkmale

Der Edit-Seite ermöglicht es die folgenden Änderungen an der Karte:

  • Ändern Sie die Karte Alias ​​für eine leicht zu merkende URL
  • Ändern Sie die Karte Sicherheitsstufe
  • Ändern Sie die Karte Titel und Beschreibung
  • Entfernen oder zeigen Daten unter der Karte
  • Aktualisieren der Daten auf der Karte

Aktualisieren der Kartendaten

Die Karte Seite bearbeiten ist ein großartiger Ort, um Daten zu einer Karte hinzufügen, die zuvor nicht geocodiert sein könnte wegen Zeitüberschreitung Fragen. Es ist auch in dem geänderten Daten können aus Ihrer Tabelle eingegeben werden. Keine Notwendigkeit, die Daten zu kopieren, einfügen gerade Ihre Tabelle Daten mit Headern, die die vorhandenen Daten übereinstimmen. Die Mapping-Koordinaten (Breite und Länge) zwischengespeichert werden, so dass nur die neuen Daten benötigt geo-kodiert werden.

Führen Sie die folgenden Schritte, um Ihre Karte zu aktualisieren:

  1. Kopieren Sie Ihre modifizierte Tabellendaten aus dem Tabellenkalkulationsprogramm
  2. Klicken Sie auf "Re-Karte"
  3. Speichern Sie die Karte, nachdem alle Änderungen zu tun
  4. Wenn es mehr als 250 neue Produkte sind, aktualisieren Sie die Edit-Seite nach dem Speichern und wiederholen Sie Schritt 1.
