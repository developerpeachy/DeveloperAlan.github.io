Adresse den einfachen Weg Geocodierung

Geocodierung ist der Vorgang des Übersetzens Adresse, Stadt, Bundesland, Postleitzahl und andere Standortinformationen in der Karte-Lage (Breitengrad und Längengrad) koordiniert. Wenn Positionsinformationen enthält bereits Breiten- und Längeninformation, gibt es keine Notwendigkeit, die Daten und eine Karte geocodieren kann in so wenig wie ein paar Sekunden durchgeführt werden.

Sie können Adressinformationen aus einer Excel-Tabelle geocodiert, indem Sie Ihre Daten einfügen in unser Service. Wir verwenden die Google Maps API für Sie die Geokodierung zu tun. Sobald alle Artikel geokodierten worden sind, werden sie auf einer Karte platziert werden. Manchmal Adressen oder Orte nicht gefunden werden kann, ein anderes Mal bekommen sie in einer falschen Stelle platziert. Wenn die Stifte an der falschen Stelle abgebildet werden, können sie bewegt werden, wenn die Karte bearbeiten.

Batch-Geokodierung Standorten.

Unser Service gibt ein rollendes Schätzung über die Zeit es dauern wird, die Geocodierung auf Basis zu beenden, wie schnell einige der vorherigen Elemente verarbeitet wurden. Im Allgemeinen kleinere Listen mit einer schnelleren Rate als größere Datenmengen verarbeiten. Googles Geo-Coder verlangsamt nach 250-Adresse. Wenn es mehr als 250 Adresse geocodiert sind, wird die Karte stoppen und eine Mitteilung vorlegen. Nachdem die Karte zu speichern, können die anderen Adressen auf dem Schütt Geocoder Seite automatisch geo-codiert sein. Dies wird zu einem Zeitpunkt 250 Adressen verarbeiten dann die Seite neu. Beim Geo-Codierung mehr als 250 Adressen abgeschlossen ist, wird der Edit-Seite angezeigt werden.
