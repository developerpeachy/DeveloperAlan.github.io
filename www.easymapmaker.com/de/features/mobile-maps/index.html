Mobile und Tablet Karten mit auf das Hotel:

Machen Sie Karten für das Handy und Tablet-Geräte mit auf der Karte Richtungen. Benutzen Sie die normalen Schritte, um eine Karte zu machen und deine Karten auf den populären mobilen Geräten und Tablets automatisch machen.

Die Geräte werden derzeit unterstützt:

  1. iPhone, iTouch und iPad
  2. Android Handys und Tablets
  3. Blackberry-Handys
  4. Windows Phones

Mobile Geräte sind auf der Karte Richtungen, und die Möglichkeit, mehrere Kategorien auf den Karten zu wählen. Aufgrund der Bildschirmgröße mobile Geräte enthalten keine Filter. IPads sind Filter, Suchfunktionen, auf der Karte Richtungen, und die Möglichkeit, mehrere Kategorien auszuwählen. Android-Tablets funktionieren ähnlich wie Handys wegen Unterstützung für bestimmte Funktionen fehlen.

Machen Sie sich bereit zu Ihrer eigenen Karte Ihrer Daten auf einer Vielzahl von Geräten zu machen! EasyMapMaker
