Karte mehrere (oder Tausende) von Standorten auf einer Karte kostenlos

Wie kann man Hunderte, wenn nicht Tausende von Standorten (Postleitzahlen, Städte, Staaten, Postleitzahlen, Ländern, Längen- und Breitengrade, etc.) auf der Karte schnell? EasyMapMaker.com wurde für diesen Zweck entwickelt wurde. Wenn Sie kopieren und Hunderte oder Tausende von Elementen einfügen, können Sie diese Karte.

Schauen Sie sich die Beispiel-Video, wie Hunderte oder Tausende von Artikeln abzubilden.

How to make a map with over 500 locations Make a map with easymapmaker.com with over 500 locations from your excel or other spreadsheet data. Go to http://www.easymapmaker.com to make your own map.

Google Maps ermöglicht die Zuordnung von bis zu 100.000 Stück auf maps.Our Dienst nicht eine Obergrenze für die Menge an Daten setzen die zugeordnet werden können, so können Sie erlaubt auf die maximale Datenmenge Karte auf.

Warum für Dienstleistungen bezahlen, die die Datenmenge begrenzen, die verwendet werden können, wenn Sie mehr thandata, dass ältere Versionen von Excel abbilden würde sogar unterstützen?

Um große Mengen von Daten zugeordnet werden, alle einzelnen Marker können nicht sichtbar. Cluster oder die Gruppierung der Nähe Marker wird unterstützt und automatisch eingestellt, wenn es mehr als 150 Artikel. Die Karten können 2500 Markierungen der Regel werden auf dem Browser ohne Performance-Probleme ab. Kombination Marker, die so dicht wie 100 in einem Cluster werden kann, ermöglicht leicht für große Daten abgebildet werden. Zoomen auf die Karte ermöglicht es, die einzelnen Marker sichtbar zu sein.

Es gibt 2 Möglichkeiten larges Datenmengen zur Karte

1. Daten mit Längen- und Breitenkoordinaten
  1. Achten Sie darauf, Ihre Daten über die erforderlichen Längen- und Breitenkoordinaten
  2. Machen Sie eine Karte auf unserer regulär oder fortgeschritten Kartenerstellung Seite
  3. Save the Map
2. Daten mit Adressen (keine Längen- und Breitenkoordinaten)

Die zweite Option ist, wenn Ihre Daten nicht Längen- und Breitengrade hat.

  1. Machen Sie eine Karte auf unseren regelmäßigen oder erweiterte Kartenherstellung Seite
  2. Speichern Sie die Karte nach 250 Koordinaten haben kartiert
  3. Klicken Sie auf den Link, der automatisch auf Masse-geo erzeugt wird, um die restlichen Daten (gelb markierten Teil in Bild)
  4. Geokodierte Stellen werden automatisch gespeichert, nachdem alle 250 Standorten verarbeitet wurden.
  5. Nachdem alle Stellen verarbeitet worden sind, wird die Bearbeitungs Karte Seite geladen keine endgültige Anpassungen vorzunehmen.
