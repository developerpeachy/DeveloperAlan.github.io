Embed Karten auf Ihrer Website

Nicht mehr den Umgang mit der komplexen Google Maps API und JavaScript. Sie können auf die Größe Karten auf Ihrer Website einbinden Sie verwenden möchten EasyMapMaker

Die obige Karte ist ein Beispiel eines eingebetteten Karte EasyMapMaker gemacht werden. Um einen eigenen eingebetteten Karte machen, machen eine normale Karte mit EasyMapMaker dann die Karte ID nehmen und es mit dem Code-Snippet verwenden unten. Die Höhe, Breite, und alle anderen Werte für die iframe kann modifiziert werden, wie Sie wollen. 

<iframe width="600" height="600"
    frameborder="0" scrolling="no"
    marginheight="0" marginwidth="0"
    src="http://www.easymapmaker.com/map/map_id"/>
</iframe>
