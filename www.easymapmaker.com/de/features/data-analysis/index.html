Visualisieren und Standort-Daten Trends zu finden

In ein anderes Element zu den typischen Tabellendaten, indem Sie auf einer Karte die Ortsdaten zu visualisieren. Karten können mit Filter erstellt werden, für eine noch bessere Analyse zu ermöglichen. Mapping, Filtern und Anzeigen von Daten ist schnell und einfach.

Das obige Beispiel zeigt Städten mit hohen mittleren Einkommen und Filterung durch eine Reihe von die Bevölkerung der Stadt aktiviert.

EasyMapMaker allows bis zu 12 Filter werden von der vorgeschobenen Karten Reiter ausgewählt. Bis zu 200 Einträge auf gefiltert werden kann. Excel macht das Hinzufügen von benutzerdefinierten reicht einfach und EasyMapMaker ermöglicht die Bereiche aus der resultierenden Daten ausgeblendet werden.

Die Daten können auch gruppiert werden (oder klassifiziert) und in einem Kreisdiagramm angezeigt, wenn Clustering aktiviert ist. Die Daten werden automatisch in Bereiche gruppiert, wenn es mehr als 10 Kategorien. Kategorisieren von Daten auf der Karte ist so einfach wie die Gruppe von Reiter aus den Kartenoptionen einstellen.
