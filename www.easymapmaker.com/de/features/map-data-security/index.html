Sichere Karten und Daten

Unser Service dauert mehrere Maßnahmen, um Ihre Daten und Karte zu schützen. Ihre Daten werden auf unseren Servern nicht hochgeladen, bis Sie ausdrücklich zu speichern Sie Ihre Karte wählen. Alle Benutzer in der Lage, unsere sichere Website zu nutzen, um ihre Karten laden oder anzeigen. https://www.easymapmaker.com/de Beim Speichern Ihrer Karte gibt es eine Option, die Karte privat zu halten oder sie der Öffentlichkeit zugänglich. Die URLs der privaten Karten werden nie eine eindeutige ID geteilt und haben die Karte zugreifen. Die Karte hat auch eine eindeutige ID ermöglicht es bearbeitet werden.

So haben die sicherste Karte gehen Sie folgendermaßen vor

  1. Besuchen Sie unsere sichere Website https://www.easymapmaker.com/de
  2. Halten Sie Ihre Karte Privat beim Speichern
  3. Ansicht und Ihre Karte auf unserer sicheren Website bearbeiten, indem Sie sicherstellen, dass es geht wenn ssl (https://www.easymapmaker.com/de)
