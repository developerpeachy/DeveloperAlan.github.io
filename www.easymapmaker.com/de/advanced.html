Speichern Karte Schließen

1.Kopieren und Einfügen von Daten

Verwenden EasyMapMaker abzubilden Excel oder andere Tabellenkalkulationsdaten auf einer Google-Karte. EasyMapMaker nimmt die Arbeit von Hand mehrere Standorte auf einer Karte aufgetragen. Der erste Schritt besteht darin, Ihre Tabellendaten einschließlich der Spaltenüberschriften in der Tabelle zu kopieren, indem Sie auf den Tisch Klick ein Textfeld zu offenbaren.

2. Set Options

Legen Sie die Optionen für Ihre Karte. Achten Sie darauf, Standortinformationen einzustellen. Standort-Informationen können viele Kombinationen der Adresse bestehen aus (Straße, Ort, Bundesland, Postleitzahl) sowie die Längen- und Breitengrade (Set explizit in separaten Spalten). Die gesamte Lage kann in einer Spalte und als Adresse festgelegt werden, oder es kann in mehreren Spalten sein. Stellen Sie andere Optionen, wenn die Daten sie enthält. Verwenden Sie erweiterte Karten Filter hinzuzufügen, fügen Sie eine linke Richtung Platte mit einem Radius Filter verstecken bestimmte Daten, laden Sie ein individuelles Logo, oder verwenden Sie benutzerdefinierte Markierungen (Pins).

3. machen Karte

Klicken Sie auf Make Karte und warten Adresse Standortdaten in Längen- und Breitengrad übersetzt werden, so dass es auf der Karte platziert werden können. Sobald die Daten auf der Karte ist, können die Stifte bewegt oder entfernt werden. Sobald die Karte sieht gut aus, speichern es. Eine E-Mail wird für Ihre Unterlagen zugesandt, die Karte in der Zukunft zu erhalten. Nachdem der Karte gespeichert ist, wird die Karte URL zur Verfügung gestellt werden. Diese URL kann auf etwas leichter zu merken geändert werden.
