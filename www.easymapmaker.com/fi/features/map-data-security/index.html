Turvallinen kartat ja tiedot

Palvelumme kestää useita toimenpiteitä suojaamaan tietoja ja kartan. Tietojasi ei ladattu palvelimillemme ennen kuin erikseen valita tallentaa kartan. Kaikki käyttäjät pystyvät käyttämään suojatun sivuston ladata tai tarkastella karttoja. https://www.easymapmaker.com/fi Kun säästää kartta on mahdollisuus pitää karttaa yksityinen tai asettaa sen julkisesti saataville. URL Yksityisten kartat koskaan jaettu, ja on ainutlaatuinen id käyttää karttaa. Kartta on myös ainutlaatuinen id mahdollistaa sen voi muokata.

Jos haluat, että varmin kartan seuraavasti

  1. Käy turvallinen sivusto https://www.easymapmaker.com/fi
  2. Pidä karttaa yksityiset tallennettaessa se
  3. Tarkastele ja muokkaa kartan suojatun sivuston varmistamalla se menee kuitenkin ssl (https://www.easymapmaker.com/fi)
