Вставить Карты на вашем сайте

Нет больше дело с комплексной Google Maps API и JavaScript. Вы можете встроить карты на вашем сайте, чтобы нужного размера с помощью EasyMapMaker

Карта выше приведен пример встроенной карты сделаны с использованием EasyMapMaker. Чтобы сделать свой собственный встроенной карты, сделать нормальную карту, используя EasyMapMaker затем взять карту-идентификатор и использовать его с фрагменте кода ниже. Высота, ширина, а также любые другие значения для фрейма может быть изменен, как вы хотите. 

<iframe width="600" height="600"
    frameborder="0" scrolling="no"
    marginheight="0" marginwidth="0"
    src="http://www.easymapmaker.com/map/map_id"/>
</iframe>
