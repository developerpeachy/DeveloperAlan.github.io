Google Maps быстро и легко сделать своими штифтами

Карта нескольких местах на карте пользовательских Google. Наша служба была разработана с использованием некоторых из новейших технологий, включая API Карт Google. Обычно делает пользовательскую карту включает утомительной работы ввода данных на карте. Мы позволяют копировать все данные и использовать API Карт Google для сопоставления данных. Мы также используем другие сложные методы, чтобы сделать данные поиск и фильтрацию, используя вещи, как перевернутые индексов (некоторые технологии за поисковиков).

Это действительно такой простой

  1. Копирование и вставка данных из таблицы в наша форма
  2. Установите соответствующие параметры (по желанию)
  3. (Необязательно для продвинутых карт) установите элементы для отображения и фильтры
  4. Нажмите "Сделать карту" (все это делается!)
