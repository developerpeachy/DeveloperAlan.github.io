Простой Карта Редактирование

Хотите обновить карту с измененными данными? Если адрес электронной почты был введен при сохранении карту, там будет будет гиперссылка для просмотра карты и другой адрес для доступа к странице, чтобы редактировать карту.

Особенности Редактировать страницу

Страница редактирования позволяет сделать следующие изменения в карту:

  • Изменить псевдоним карту для легкого помнить URL
  • Изменение уровня безопасности карта
  • Изменить название карты и описание
  • Удалить или показать данные ниже карте
  • Обновление данных о карте

Обновление картографических данных

Страница Карта Редактировать это прекрасное место для добавления данных в карте, которые не могли быть ранее геокодированы за задержки вопросам. Это также, где модифицированные данные могут быть введены из таблицы. Нет необходимости, чтобы скопировать данные, просто вставьте данные таблиц с заголовками, которые соответствуют имеющиеся данные. Координаты отображение (широта и долгота) кэшируются, поэтому только новые данные должны быть геокодированы.

Выполните следующие действия, чтобы обновить карту:

  1. Копирование измененных данных таблицы из таблицы программы
  2. Нажмите "Re-Map"
  3. Сохранить карту после выполнения каких-либо изменений
  4. Если есть новые пункты более 250, обновите страницу редактирования после сохранения его и повторите шаг 1.
