시각화 및 동향을 찾기 위해 위치 데이터를지도

지도에서 위치 데이터를 시각화하여 일반적인 스프레드 시트 데이터에 다른 요소를 추가합니다. 지도는 더 나은 분석을 할 수 있도록 필터를 만들 수 있습니다. 매핑, 필터링 및보고 데이터를 빠르고 간단합니다.

위의 예는 도시의 인구의 범위에 의해 활성화 높은 평균 소득과 필터링 도시를 보여줍니다.

EasyMapMaker a(12) 필터까지 llows는 고급 맵 탭에서 선택합니다. 최대 200 항목에 필터링 할 수 있습니다. Excel에서 사용자가 쉽게 범위를 추가하게하고, EasyMapMaker는 범위는 결과 데이터에서 숨길 수 있습니다.

데이터는 그룹화 (또는 분류) 및 클러스터링을 사용하는 경우 원형 차트에 표시 할 수 있습니다. 10 개 이상의 카테고리가있는 경우 데이터는 자동 범위로 그룹화됩니다. 지도 데이터를 분류하는지도 옵션 탭에서 그룹을 설정하는 것만 큼 쉽습니다.
