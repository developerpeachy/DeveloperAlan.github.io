데이터의 라지 양을지도하는 2 가지 방법이 있습니다

위도 및 경도 좌표 1. 데이터
  1. 데이터가 필요한 위도와 경도 좌표를 가지고 있는지 확인하십시오
  2. 지도에서 확인 우리 정규병 또는 많은 지도를 만드는 페이지
  3. Save the Map
주소 (더 위도와 경도 좌표 없음) 2. 데이터

두 번째 옵션은 데이터가 위도와 경도 데이터가없는 경우입니다.

  1. 우리의 일반 또는 고급지도 제작 페이지에서지도를 확인
  2. 250 좌표가 매핑 된 후지도를 저장
  3. 자동으로 대량 지​​역에 남아있는 데이터를 생성 된 링크를 클릭 (노란색 이미지의 부분을 강조)
  4. 모든 250 위치가 처리 된 후 지오 코딩 위치는 자동으로 저장됩니다.
  5. 모든 위치가 처리 된 후, 편집지도 페이지는 최종 조정을 위해로드됩니다.
