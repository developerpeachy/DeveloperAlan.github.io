안전지도 및 데이터

우리의 서비스는 데이터와지도를 보호하기 위해 여러 조치를합니다. 명시 적으로지도를 저장하도록 선택할 때까지 데이터는 우리의 서버에 업로드되지 않습니다. 모든 사용자는 업로드 또는지도를 볼 보안 사이트를 사용할 수 있습니다. https://www.easymapmaker.com/ko 지도를 저장시 개인지도를 유지하거나 공개적으로 사용할 수 있도록하는 옵션이 있습니다. 개인지도의 URL을 공유하지 않고지도에 액세스 할 수있는 고유 ID가 않습니다. 지도는 또한 편집 할 수 있도록 고유 한 ID가 있습니다.

가장 안전한지도 다음 단계를 수행해야합니다

  1. 보안 사이트 방문 https://www.easymapmaker.com/ko
  2. 를 저장할 때 개인지도를 유지
  3. 보기 및 확인은 SSL 불구하고 진행하여 보안 사이트에지도를 편집 (https://www.easymapmaker.com/ko)
