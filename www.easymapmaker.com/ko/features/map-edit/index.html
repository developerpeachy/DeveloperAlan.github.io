간단한지도 편집

수정 된 데이터와지도를 업데이트하려면? 이메일 주소가지도를 저장하는 동안 입력 된 경우,지도를 편집 할 페이지에 액세스지도를 다른 URL을 볼 수있는 URL이있을 것이다 것입니다.

페이지 편집 기능

편집 페이지지도를 다음과 같이 변경 할 수 있습니다 :

  • 기억하기 쉬운 URL에 대한지도 별명을 수정
  • 지도 보안 수준을 변경
  • 지도의 제목 및 설명 수정
  • 제거하거나지도 아래 데이터를 표시
  • 지도 데이터를 업데이트

지도 데이터 업데이트

지도 편집 페이지 인해 시간 제한 문제 이전에 지오 코딩 수 없습니다지도에 데이터를 추가 할 수있는 좋은 장소입니다. 수정 된 데이터가 스프레드 시트에서 입력 할 수있는 곳도 있습니다. 데이터를 복사 할 필요는 기존의 데이터와 일치 헤더 스프레드 시트 데이터를 붙여 넣기합니다. 매핑 좌표 (위도 및 경도) 따라서 만 새로운 데이터가 지리적으로 구분 될 필요가 캐시된다.

지도를 업데이트하려면 다음 단계를 따르세요

  1. 스프레드 시트 프로그램에서 수정 된 스프레드 시트의 데이터를 복사
  2. 클릭 "다시지도"
  3. 수정을 수행 한 후지도를 저장
  4. 250 개 이상의 새로운 항목이있는 경우이를 저장 한 후 편집 페이지를 새로 고침하고 1 단계를 반복합니다.
