검색 및지도 필터

당신은 특정 주소 또는 다른 위치에 가장 가까운 위치를 찾고 계십니까? 또한, 검색 상자를 사용하여 다른 위치에 가장 가까운 위치를 쉽게 찾을 수 있습니다. 위치와 엔터를 입력하거나 검색 아이콘을 클릭하고 서비스가 자동으로 가장 가까운 위치로지도를 이동합니다.

뿐만 아니라 데이터 필터링

검색 상자는 또한 필터 상자로 기능한다. 당신이 뭔가를 찾고있다 꽤 그것이 무엇인지 기억 나지 않는 경우, 그냥 검색 창에 입력하고 몇 문자 후 항목의 목록이 표시됩니다. 여러 매핑 된 위치와 같은 도시 이름 등의 항목을 선택하면,지도 항목 만 표시 확대됩니다. 스프레드 시트에서 데이터의 대부분은 자동으로 유사한 데이터에 쉽게 필터링하고, 검색 범위에 입력됩니다. 이것은 서로 다른 데이터 열을 필터링 할 수 있습니다 고급 맵에서 생성되는 미리 정의 된 필터와 혼동하지 마십시오. 그러나 일부 상황들은 이러한 도시는 검색 창에 여과하고, 필터가 설치되어 도시의 경우와 같이 동일한 방식으로 기능 할 수있다.

그룹에 의해 필터 (테마 또는 카테고리)

최대 10 개의 다른 마커 아이콘으로 그룹 데이터. 클러스터링 그룹과 함께 사용하는 경우, 파이 차트는 선택한 그룹의 밀도를 보여줍니다.
