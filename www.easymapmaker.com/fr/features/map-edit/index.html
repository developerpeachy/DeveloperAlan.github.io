Simple Plan Édition

Vous voulez mettre à jour votre carte avec les données modifiées? Si une adresse électronique a été conclu lors de l'enregistrement de la carte, il y aura aura une URL pour afficher la carte et une autre URL pour accéder à une page pour modifier la carte.

Modifier la page Caractéristiques

La page d'édition permet les modifications suivantes à la carte:

  • Modifier la carte alias pour un facile à retenir URL
  • Modifier le niveau plan de sécurité
  • Modifier le titre de la carte et description
  • Supprimer ou afficher les données ci-dessous la carte
  • Mettre à jour les données sur la carte

Mise à jour des données cartographiques

La page Carte Edit est un endroit idéal pour ajouter des données à la carte qui pourrait ne pas être déjà géocodées en raison de problèmes temporisation. Il est également lorsque les données modifiées peuvent être entrées à partir de votre feuille de calcul. Pas besoin de copier les données, il suffit de coller vos données de feuille de calcul avec les en-têtes qui correspondent à des données existantes. Les coordonnées de mapping (latitude et longitude) sont mis en cache, de sorte que seules les nouvelles données devront être géocodées.

Suivez les étapes ci-dessous pour mettre à jour votre carte:

  1. Copier vos données de feuille de calcul modifiés depuis votre tableur
  2. Cliquez sur "Re-carte"
  3. Enregistrer la carte après avoir fait des modifications
  4. S'il y a plus de 250 nouveaux éléments, actualisez la page d'édition après l'avoir enregistré et répétez l'étape 1.
