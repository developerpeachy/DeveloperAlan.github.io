Intégration de cartes sur votre site

Pas plus traiter avec le complexe API Google Maps et JavaScript. Vous pouvez intégrer des cartes sur votre site à la taille que vous voulez en utilisant EasyMapMaker

La carte ci-dessus est un exemple d'une carte intégrée fait en utilisant EasyMapMaker. Pour faire votre propre carte intégrée, faire une carte normale en utilisant EasyMapMaker puis prendre la carte ID et l'utiliser avec l'extrait de code ci-dessous. La hauteur, la largeur et toutes les autres valeurs de l'iframe peuvent être modifiés que vous le souhaitez. 

<iframe width="600" height="600"
    frameborder="0" scrolling="no"
    marginheight="0" marginwidth="0"
    src="http://www.easymapmaker.com/map/map_id"/>
</iframe>
