Maps Recherche avancée

Avez-vous jamais eu envie de porter la capacité à filtrer les données dans Excel pour vous Maps? Nos "Plans avancées" vous permettent de faire exactement cela. L'onglet Carte avancée permet les mêmes fonctionnalités que "Plans Facile" et plus encore. Personnalisez votre carte en ajoutant vos propres images et logo. Si vous voulez voir les données qui sont contenues dans une région de la carte, la fonction de données d'exportation fournit cette.

Regardez la vidéo pour faire une carte avec les filtres, les repères de mesure, un logo, l'exportation de données, itinéraire, et un filtre de localisation.

Plans avancées permettent une plus grande personnalisation:

  1. Ajouter des images personnalisées
  2. Ajouter les filtres (Sélection multiple, seule sélection, nombre d'éléments)
  3. Cacher données

Ajouter des images personnalisées (Voir exemple de carte )

Ajouter une image personnalisée ou le logo de votre carte en les téléchargeant. Vous pouvez télécharger jusqu'à 50 images à 1 fois avec une taille allant jusqu'à 10 Ko chacun. Pour utiliser plusieurs images, le groupe doit être défini. Les images peuvent être mises en correspondance avec les différents groupes (ou catégories). L'image ci-dessous est un exemple de configuration d'un groupe (pour être utilisé avec les images dans la photo ci-dessus).

Ajouter des filtres d'un simple clic.

Masquer les éléments que vous ne voulez pas que les utilisateurs voient

Our Maps Recherche avancée vous permettent d'avoir des données consultables et filtrables sur la carte mais pas affichées pour les téléspectateurs de la carte. Coordonnées peuvent maintenant être facilement cachés, ou des articles qui sont filtrés ou groupés sur Are ne peuvent être visibles avec les données source et caché des utilisateurs. Cela est aussi facile que de sélectionner les éléments à cacher. Par défaut, tous les éléments sont visibles.

La carte ci-dessus est un exemple à l'aide des groupes et des filtres

Plans avancés comprennent également la possibilité de masquer tous les en-têtes de l'étiquette pour la description de marqueur par de-sélectionner la case à cocher "Afficher étiquette". Cette option doit être utilisée avec prudence, sauf les données est évidente pour l'utilisateur.
