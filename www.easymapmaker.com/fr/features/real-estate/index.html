Carte annonces immobilières

(EasyMapMaker) Notre service a été conçu avec des annonces immobilières à l'esprit. Si vous avez une liste des propriétés dans une base de données ou un tableur, vous pouvez facilement importer les données dans notre service pour une utilisation. Vous pouvez faire des filtres personnalisés pour des choses comme le nombre de chambres, salles de bains, plages de superficie, date de construction. La combinaison de notre service à la puissance d'Excel permet à vos cartes de dynamique avec seulement quelques clics.

Vous pouvez mapper les ventes antérieures, des comparaisons, des immeubles locatifs, les types d'annonces (les ventes à découvert, saisies, à vendre par le propriétaire, OER, etc). Notre service prendra vos données avec les adresses, vous permettre de sélectionner vos filtres sur les cartes de pointe et la carte sera générée et partageable à qui vous voulez.(EasyMapMaker)

Vous pouvez ajouter des liens vers un site Web externe pour tous les détails de la propriété et même mettre un lien vers l'image pour les utilisateurs de voir quand ils cliquent sur un marqueur. De plus la carte est visible sur les appareils mobiles (moins certaines caractéristiques en raison de la taille de l'écran).

Voir un exemple d'annonces immobilières:

Pour faire une carte comme vous-même, téléchargez le fichier d'exemple et suivez les instructions: ( Télécharger )

  1. #Copy and paste data from the spreadsheet into the advanced map form
  2. Cliquez sur "Définir les options"
  3. Définir le groupe d'être FOURCHETTE (Voir l'image)
  4. Dans la section "Choisir les éléments à afficher dans la description de marqueur», cacher la gamme de prix
  5. Dans le "Choisir jusqu'à 12 articles de filtrer sur" section, assurez-vous HOME TYPE, lits, et les bains sont définies dans la section de filtre (Voir l'image)
  6. Cliquez "Faire de carte" et vous êtes d'or après avoir sauvé la carte!
Alors que EasyMapMaker.com est capable de générer des plages à partir des données, pour obtenir la plage désirée, il est préférable d'utiliser Excel (comme cela a été fait dans le fichier de l'échantillon).
