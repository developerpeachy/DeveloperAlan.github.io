地図にあなたのExcelやスプレッドシートのデータを回し

EasyMapMaker 当社のサービスは、次の2つの一般のアプリケーション（ExcelスプレッドシートのデータとGoogleマップ）を埋め、1にそれらを回すために設計されました。マッピングアドレスまたはロケーションベースのデータによって次のレベルにあなたのExcelや他のスプレッドシートアプリケーションデータを取ります。国、州（および州）、都市、郵便番号（郵便）コード、道路、および特定の住所：次を含む、のいずれかの場所多くの種類をマッピングすることができます。より具体的なアドレスは、より多くの微調整メーカピンがマップ上に配置されます。あなたは、データをコピーして貼り付けることができた場合、あなたはそれをマッピングすることができます。

フィルタリング、データ隠蔽、方向テキスト、距離ベースのフィルタリング、または地図上にデータをエクスポートが望まれる場合、高度なマップを使用します。

フィルター
私たちのregulatデータフィルタは、上でフィルタリングする200項目まで可能にし、最大12のフィルタが存在することができます。複数選択は、彼らが空白になっていない限り、すべてのすべてのユニークなアイテムがオンにフィルタリングするフィルタ。
Naviagation（行き方）との距離のフィルタリング
左手方向（より多くのオプション）ナビは、地図だけでなく、別の1点から順方向による詳細なターンで方向を示すためのオプションが含まれています。近隣のフィルタは、（必要な場合）の円が表示されることを可能にするだけでなく、位置からの距離に基づいてフィルタ。その位置からの距離は、地図の下のデータで表示することができます。

複雑なマップが簡単に行うことができます

  1. スプレッドシートからデータをコピー・アンド・ペーストするに EasyMapMaker
  2. （オプション）適切なオプションを設定します
  3. （高度なマップ用オプション）表示する項目とフィルタを設定します
  4. 「マップを作成する」をクリックします（すべてやりました！）

地図上に置くと、独自のカスタムフィルタを追加することによって、次のレベルにあなたのエクセル（表計算）のデータを取ります。
