シンプルな地図の編集

変更されたデータを使用してマップを更新してみませんか？地図を節約しながら、電子メールアドレスが入力された場合は、マップを編集するには、ページにアクセスするには、マップや別のURLを表示するためのURLが存在することになるでしょう。

ページの編集機能

編集ページは、マップに次の変更を許可します：

  • 覚えやすいURLに対するマップの別名を変更します
  • 地図のセキュリティレベルを変更します
  • 地図のタイトルと説明を変更します。
  • 削除するか、地図の下のデータを表示
  • 地図上のデータを更新します

地図データを更新

地図Editページが原因でタイムアウトの問題に以前にジオコーディングすることができませんでしたマップにデータを追加するには絶好の場所です。変更されたデータは、スプレッドシートから入力することができる場所それはまたです。データをコピーする必要はありません、単に既存のデータと一致しているヘッダーを使用してスプレッドシートのデータを貼り付けます。マッピング座標（緯度と経度）がキャッシュされるので、新しいデータのみがジオコーディングする必要があります。

あなたのマップを更新するには、以下の手順に従います。

  1. スプレッドシートプログラムから変更したスプレッドシートのデータをコピーします。
  2. 「再マップ」をクリックしてください
  3. 変更を行った後にマップを保存
  4. 250以上の新しいアイテムがある場合は、それを保存した後、編集ページを更新し、ステップ1を繰り返します。
