セキュアなマップとデータ

当社のサービスは、あなたのデータとマップを保護するためのいくつかの措置をとります。あなたが明示的にマップを保存することを選択するまで、あなたのデータは、当社のサーバーにアップロードされていません。すべてのユーザーがアップロードしたり、マップを表示するには、弊社の安全なサイトを使用することができます。 https://www.easymapmaker.com/ja あなたのマップを保存すると、プライベートマップを維持するか、それが公に利用できるようにするオプションがあります。プライベートマップのURLが共有されず、マップにアクセスするための一意のIDを持っているん。マップはまた、編集することを可能にする固有のIDを有しています。

最も安全マップは次の手順に従ってください持っているために、

  1. 安全確実な弊社のサイトをご覧ください https://www.easymapmaker.com/ja
  2. それを保存するときに、プライベートマップを保管してください
  3. 必ずそれは、SSLものの行く行うことで、弊社の安全なサイト上であなたのマップを表示し、編集します (https://www.easymapmaker.com/ja)
