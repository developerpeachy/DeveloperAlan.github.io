Perguntas comuns e suas respostas (FAQ)

Quantos endereços posso adicionar a um mapa?
Você pode adicionar mais de 10.000 locais únicos por dia no seu mapa. Levará horas para geocode-los, mas é factível. Se você já tem a coordenadas de latitude e longitude os dados podem ser adicionados em menos de 5 segundos.
Como faço para adicionar mais de 250 itens para um mapa?

Você pode adicionar milhares de locais de um dia para o seu mapa. O limite é de pelo menos 10.000 locais.

Apenas 250 endereços em um momento pode ser processado, depois de 250 localizações geo-codificação vai para um rastreamento.

Confira o vídeo de exemplo para saber como mapear centenas ou milhares de itens.
How to make a map with over 500 locations Make a map with easymapmaker.com with over 500 locations from your excel or other spreadsheet data. Go to http://www.easymapmaker.com to make your own map.

Uma vez que 250 endereços terminar, salve o seu mapa e um link para bulk-geocode será exibida. Clique neste link e os locais serão automaticamente geo-codificados em lotes de 250 e salva automaticamente para você. Uma vez que todos os endereços têm ser geo-codificados a página de edição será carregado. Com licença gratuita do Google pode demorar mais de uma hora para processar 5.000 endereços.

O que se passa, por que é o seu serviço gratuito?
Nosso serviço é 100% gratuito. Nós não pagamos google nada para usar sua API. Nossa equipe faz dinheiro através de seus trabalhos do dia. Se você quiser doar você pode.
Posso substituir o logotipo EasyMapMaker no meu mapa?
Sim. Adicione o seguinte parâmetro para o seu mapa: logo = <url codificado>. Você pode codificar o seu url usando o nosso ferramenta de codificação de url .

Um exemplo é a seguinte:

http://www.easymapmaker.com/map/58890adb2af63457f31769e607d7501f?logo=http%3A%2F%2Fwww.easymapmaker.com%2Fcluster_images%2Fsemi_opaq%2F45_blue.png

Outra forma de adicionar um logotipo personalizado é usar mapas avançados e fazer o upload de um logotipo na seção Personalização.

O tamanho da imagem padrão é 45x45 pixels ea substituição deve idealmente ser o mesmo.

Posso alterar a fonte ou adicionar um estilo personalizado para o meu mapa?
Sim. Adicione o seguinte parâmetro para o seu mapa: css = <url codificado>. Você pode codificar o seu url usando o nosso ferramenta de codificação de url. Um exemplo é a seguinte: http://www.easymapmaker.com/map/58890adb2af63457f31769e607d7501f?css=http%3A%2F%2Fwww.easymapmaker.com%2Fcssoverride%2Fhideimg.css
Como posso atualizar ou excluir um mapa existente?
Se você entrar com um endereço de e-mail quando você salvou o mapa. Ele deveria ter incluído um link para editar o mapa. O mapa pode também ser retirado a partir dessa página. Às vezes, os e-mails vai para a pasta de spam ou junk mail, para que possa verificar lá também. Se você quer uma lista de todos os seus mapas para um endereço de e-mail, vá para o http://www.easymapmaker.com/findMyMaps

Como posso ver meus dados no Google Maps?
Se você gerou o kml, você pode importá-lo em um mapa do Google (hospedado pelo Google).

Existe uma maneira de não mostrar uma lista de dados abaixo do mapa?
Sim, você pode desligar os dados na parte inferior da página. Se você vai para a página de edição (deve ser em um e-mail enviado depois de salvar o mapa), há seção um "Mapa de layout '. Você tem a opção de desligar os dados que mostram abaixo da página.
Posso fazer um mapa sem um endereço?
Sim. Se você tem um local latitude e longitude, aqueles podem ser mapeados rapidamente. Se você tiver apenas parte de um endereço, como um código postal ou cidade, estado, em seguida, aqueles que podem ser mapeados bem.

Como posso atribuir cores diferentes para diferentes produtos no meu conjunto de dados para que eles possam ser identificados no mapa.?
As cores são atribuídos automaticamente com base na coluna que você definir como o grupo. Existem até 10 (20 para mapas avançados) cores diferentes que são usados. Se uma coluna tem mais de 10 (20 para mapas avançados) valores exclusivos diferentes, eles serão agrupados em faixas. É melhor para definir a faixa-se e usar isso para o grupo (se você tem mais de 10 valores exclusivos em uma coluna).
Como posso obter gráficos de pizza meu mapa?
Os gráficos de pizza são ativados automaticamente quando você ligar o agrupamento e definir um grupo do seu mapa (veja a pergunta acima para grupos).
Quanto tempo o meu mapa permanecer ativo?
Mapas serão permanecem ativas, a menos que não foram modificados ou visualizados nos últimos 90 dias. Eles podem ser excluídos a qualquer momento antes dessa pela pessoa que fez o mapa.
Como posso imprimir o meu mapa, guardá-lo como uma imagem ou um arquivo pdf?

Para salvar seu mapa como um arquivo pdf, use o Google Chrome ou Safari imprimir o mapa e guardá-lo como um arquivo pdf em vez de imprimi-lo.

Você pode salvar seu mapa como uma imagem, tendo uma captura de tela do seu mapa (alt + print screen funciona normalmente, ou tentar print screen ctrl +). Depois de ter uma captura de tela, você deve ser capaz de colá-lo em uma ferramenta de edição de imagem como o paint.net. O tamanho do mapa pode ser tão grande quanto a janela do navegador web se você não incorporá-lo. Se você salvar seu mapa como uma imagem e quer convertê-lo em PDF, você pode usar o nosso para ferramenta de conversão de pdf
