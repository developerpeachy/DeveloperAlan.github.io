Roteiro anúncios de imóveis

(EasyMapMaker) Nosso serviço foi projetado com anúncios imobiliárias em mente. Se você tem uma lista de propriedades em um banco de dados ou planilha, você pode facilmente importar os dados para o nosso serviço para uso. Você pode fazer filtros personalizados para coisas como número de quartos, casas de banho, gamas de metragem quadrada, data de construção. Combinando o nosso serviço com o poder do Excel permite que os seus mapas para dinâmico, com apenas alguns cliques.

Você pode mapear as vendas anteriores, comparações, imóveis para alugar, os tipos de anúncios (vendas a descoberto, arrestos, à venda pelo proprietário, REOs, etc). Nosso serviço irá levar os seus dados com endereços, permitem selecionar seus filtros em mapas avançados e o mapa será gerado e compartilhável para quem quiser.(EasyMapMaker)

Você pode adicionar links para um site externo para todos os detalhes da propriedade e até mesmo colocar um link para a imagem para os usuários para ver quando clicarem em um marcador. Além disso, o mapa pode ser visualizado em dispositivos móveis (menos algumas características devido ao tamanho da tela).

Para fazer um mapa como este mesmo, baixar o arquivo de amostra e siga as instruções: ( baixar )

  1. #Copy and paste data from the spreadsheet into the advanced map form
  2. Clique em "Opções"
  3. Definir o grupo a ser Faixa de preço (imagem mostra)
  4. Na seção "Escolher itens para mostrar na descrição do marcador", ocultar a faixa de preço
  5. Na "Escolha até 12 itens para filtrar em" seção, certifique-se Tipo de casa, camas, e banhos são definidas na seção Filtro (imagem mostra)
  6. Clique em "Criar Map" e você são de ouro depois de salvar o mapa!
Enquanto EasyMapMaker.com é capaz de gerar varia de dados, para obter o intervalo desejado, é melhor usar o Excel (como foi feito no arquivo de exemplo).
