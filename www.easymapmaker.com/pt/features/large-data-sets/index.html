múltipla Mapa (ou milhares) de locais em um mapa de graça

Como você colocar centenas, se não milhares de locais (códigos postais, cidades, estados, CEPs, países, coordenadas de latitude e longitude, etc.) no mapa rapidamente? EasyMapMaker.com foi concebido para este propósito. Se você pode copiar e colar centenas ou milhares de itens, você pode mapeá-los.

Confira o vídeo de exemplo para saber como mapear centenas ou milhares de itens.

How to make a map with over 500 locations Make a map with easymapmaker.com with over 500 locations from your excel or other spreadsheet data. Go to http://www.easymapmaker.com to make your own map.

Google Maps permite mapeamento de até 100.000 itens no maps.Our serviço não colocar um limite na quantidade de dados que podem ser mapeados, para que possa mapear até a quantidade máxima de dados permitida.

Por que pagar por serviços que limitam a quantidade de dados que pode ser usado quando você pode mapear mais thandata que as versões mais antigas do Excel sequer suporta?

A fim de que grandes quantidades de dados a ser mapeada, todos os marcadores individuais não pode ser visível. Cluster ou o agrupamento de marcadores nas proximidades é suportado e definir automaticamente quando há mais de 150 itens. Os mapas geralmente podem exibir 2500 marcadores sem problemas de desempenho, dependendo do browser. A combinação de marcadores que podem ser tão denso como 100 para um cluster facilmente permite a grande quantidade de dados a ser mapeada. Zoom sobre o mapa permite que os marcadores individuais para ser visível.

Existem 2 maneiras de mapear valores larges de dados

1. Os dados com coordenadas de latitude e longitude
  1. Certifique-se de seus dados tem as necessárias latitude e longitude
  2. Faça um mapa em nosso regular ou avançado página do mapa tomada
  3. Save the Map
2. Os dados com endereços (sem latitude e longitude)

A segunda opção é quando os seus dados não tem dados de latitude e longitude.

  1. Faça um mapa na nossa página de mapa tomada regular ou avançado
  2. Salvar o mapa depois de 250 coordenadas foram mapeados
  3. Clique no link que é gerado automaticamente para bulk-geo os dados restantes (amarelo destaque porção na imagem)
  4. locais geocodificadas são salvas automaticamente a cada 250 locais foram processadas.
  5. Uma vez que todos os locais tenham sido processados, a página mapa edição será carregado para fazer os ajustes finais.
