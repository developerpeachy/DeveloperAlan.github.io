Mapas avançados

Alguma vez você já quis trazer a capacidade de filtrar os dados no Excel para você Maps? Nossos "Mapas avançadas" permitem que você faça exatamente isso. A guia Mapa Avançada permite a mesma funcionalidade que "Mapas Easy" e muito mais. Personalize o seu mapa adicionando suas próprias imagens e logotipo. Se você quiser ver os dados que estão contidos em uma região do mapa, o recurso de exportação de dados fornece isso.

Confira o vídeo para fazer um mapa com filtros, pinos personalizados, um logotipo, exportação de dados, direcções de condução e um filtro de localização.

Mapas avançados permitem mais personalização:

  1. Adicionar imagens personalizadas
  2. adicionar filtros (Multiselect, seleção única, a contagem de item)
  3. Ocultar dados

Adicionar imagens personalizadas (Ver mapa de exemplo )

Adicionar uma imagem personalizada ou o logotipo seu mapa, carregando-os. Você pode enviar até 50 imagens em 1 hora com um tamanho até 10KB cada um. Para utilizar várias imagens, o grupo precisa ser definido. As imagens podem ser mapeadas para os diferentes grupos (ou categorias). A imagem abaixo é um exemplo de configuração de um grupo (a ser usado com as imagens na foto acima).

Adicionar filtros com um simples clique.

Ocultar itens que você não quer que os usuários vejam

Our Mapas avançados permitir que você tenha dados consultáveis ​​e filtráveis ​​no mapa, mas não exibidas para os telespectadores do mapa. Endereço podem agora ser facilmente escondido, ou itens que são filtrados ou agrupados são apenas pode ser visível com os dados de origem e ocultos dos usuários. Isto é tão fácil como selecionar os itens para se esconder. Por padrão, todos os itens são visíveis.

O mapa acima é uma amostra usando grupos e filtros

Mapas avançados também incluem a opção para ocultar todos os cabeçalhos da etiqueta para a descrição do marcador pelo de-selecionando a caixa "show label". Esta opção deve ser utilizada com precaução, a menos que os dados é evidente para o utilizador.
