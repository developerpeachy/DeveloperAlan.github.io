Seguros mapas e dados

O nosso serviço adota diversas medidas para proteger seus dados e mapa. Seus dados não são enviados para nossos servidores até que você escolha explicitamente para salvar seu mapa. Todos os usuários são capazes de usar o nosso site seguro para carregar ou visualizar seus mapas. https://www.easymapmaker.com/pt Ao salvar seu mapa existe uma opção para manter o mapa privado ou torná-lo disponível ao público. As URLs de mapas privados nunca são compartilhados e ter um ID único para acessar o mapa. O mapa também tem uma identificação única que lhe permite ser editado.

Para que o mapa mais seguro siga estes passos

  1. Visite o nosso site seguro https://www.easymapmaker.com/pt
  2. Mantenha o seu mapa privado quando salvá-lo
  3. Ver e editar o seu mapa no nosso site seguro, certificando-se ele vai embora SSL (https://www.easymapmaker.com/pt)
