Mapas de incorporar em seu site

Não mais lidar com o complexo de API do Google Maps e JavaScript. Você pode incorporar mapas em seu site para o tamanho desejado usando EasyMapMaker

O mapa acima é um exemplo de um mapa incorporado feita usando EasyMapMaker. Para fazer o seu próprio mapa incorporado, fazer um mapa normal usando EasyMapMaker em seguida, tomar o id mapa e usá-lo com o trecho de código abaixo. A altura, largura e quaisquer outros valores para o iframe pode ser modificado como você deseja. 

<iframe width="600" height="600"
    frameborder="0" scrolling="no"
    marginheight="0" marginwidth="0"
    src="http://www.easymapmaker.com/map/map_id"/>
</iframe>
