Visualizar e mapear os dados de localização para encontrar tendências

Adicionar outro elemento para os dados da planilha típicos, visualizando os dados de localização em um mapa. Os mapas podem ser criados com filtros para permitir a análise ainda melhor. Mapeamento, filtragem e visualização de dados é rápido e fácil.

O exemplo acima mostra cidades com altos rendimentos medianos e filtragem ativado por uma série de a população da cidade.

EasyMapMaker allows até 12 filtros para ser seleccionado a partir do separador mapas avançada. Até 200 itens pode ser filtrado. Excel faz a adição personalizada varia fácil, e EasyMapMaker permite que os intervalos para ser escondido de os dados resultantes.

Os dados também podem ser agrupadas (ou categorizadas) e exibidas em um gráfico de pizza, quando o agrupamento está activado. Os dados são agrupados automaticamente em intervalos, se houver mais de 10 categorias. Categorizar os dados no mapa é tão fácil como a criação do grupo de guia das opções de mapa.
