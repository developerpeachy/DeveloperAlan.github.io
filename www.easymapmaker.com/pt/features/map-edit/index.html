Simples Mapa Edição

Quer actualizar o seu mapa com os dados modificados? Se um endereço de e-mail foi digitado ao salvar o mapa, haverá será uma url para ver o mapa e outra url para acessar uma página para editar o mapa.

Características Editar Página

A página de edição permite que as seguintes alterações no mapa:

  • Modificar o mapa alias para um fácil de lembrar url
  • Alterar o nível de segurança mapa
  • Modifique o mapa de Título e Descrição
  • Remover ou mostrar dados abaixo do mapa
  • Actualizar os dados no mapa

Atualizando os dados do mapa

A página Map Edit é um ótimo lugar para adicionar dados para o mapa que não poderia ser previamente geo-codificados devido a problemas de tempo limite. É também onde os dados modificados podem ser inseridos a partir de sua planilha. Não há necessidade de copiar os dados, basta colar dados de sua planilha com cabeçalhos que correspondem os dados existentes. As coordenadas de mapeamento (latitude e longitude) são armazenados em cache, portanto, apenas os novos dados terá de ser geo-codificada.

Siga os passos abaixo para actualizar o seu mapa:

  1. Copiar dados de sua planilha modificados a partir do seu programa de planilha
  2. Clique em "Re-Map"
  3. Salvar o mapa depois de fazer quaisquer modificações
  4. Se houver mais de 250 novos itens, atualize a página de edição após salvá-lo e repita o passo 1.
